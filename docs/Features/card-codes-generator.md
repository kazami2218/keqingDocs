## Code Filtering

![Card codes generator](/img/features/filter.png)

### How to use
- Run the `kc` command
- React with 🔍 to use this feature.​​

You can also filter the codes that you want to copy, get all the codes, get all untagged codes, or get under 100 wish card codes.
